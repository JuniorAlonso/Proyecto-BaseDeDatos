# Market "La Estancia" - Seguridad y Auditoría en Base de Datos

Proyecto funcional para el **Tema 6: Seguridad en bases de datos: usuarios, roles, permisos y auditoría**. Implementa PostgreSQL 15 (Docker) + Spring Boot (Java 25) + Frontend Web responsivo.

---

## 🚀 Inicio Rápido

### Paso 1: Iniciar Base de Datos (PostgreSQL)
Asegúrate de tener Docker activo y ejecuta en el directorio raíz del proyecto:
```bash
docker-compose up -d
```

### Paso 2: Iniciar Servidor Web (Spring Boot)
En el directorio raíz del proyecto, ejecuta:
```bash
.\mvnw spring-boot:run
```

### Paso 3: Abrir Aplicación
Dirígete a:
👉 **[http://localhost:8080/](http://localhost:8080/)**
